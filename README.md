# Автономный робот для распознавания дорожных знаков


## 📖 Обзор проекта
Данный проект представляет собой автономную мобильную платформу, управляемую связкой **Raspberry Pi 4** и **Arduino**. Основная задача робота — навигация в реальном времени с использованием компьютерного зрения (Computer Vision) для распознавания дорожных знаков и принятия решений на их основе.

Проект сочетает в себе высокоуровневую обработку видеопотока на Linux-системе и низкоуровневое управление приводами на микроконтроллере.


## 🛠 Технические характеристики и компоненты

### Электроника
  * **Вычислительный узел:** Raspberry Pi 4 (2GB RAM) — обработка OpenCV и логика принятия решений.
  * **Контроллер периферии:** Arduino Nano/Uno — управление моторами и сбор данных с датчиков.
  * **Зрение:** Raspberry Pi Camera Module (или совместимая USB-камера).
  * **Навигация:** IMU-сенсор (гироскоп, акселерометр, компас) для точного позиционирования и контроля угла поворота.

### Силовая часть
  * **Драйвер моторов:** Подбирается под мощность используемых двигателей (например, L298N или TB6612FNG).
  * **Питание:** Аккумуляторная сборка 12V (рекомендуемая емкость 2–3 Ач) с понижающими модулями для питания электроники.


## ⚙️ Особенности реализации

### 1\. Гибридная архитектура управления
Система разделена на два уровня:
1.  **High-Level (Python):** Запускается на Raspberry Pi. Отвечает за захват кадров, работу нейросети или алгоритмов обработки контуров для поиска знаков.
2.  **Low-Level (C++):** Arduino получает команды от Raspberry Pi через Serial-интерфейс и обеспечивает плавное движение, а также стабилизацию курса по IMU.

### 2\. Интегрированная среда разработки (CI/CD)
Одной из ключевых фич проекта является удобство деплоя:
  * Настройка **VS Code Remote Development** для работы напрямую на Raspberry Pi.
  * Реализована возможность прошивки Arduino «на лету» прямо из среды разработки или при обновлении кода в репозитории (Git-driven workflow).
