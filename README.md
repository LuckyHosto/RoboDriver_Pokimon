Привет! Робототехники и DIY-энтузиасты, этот проект посвящён маленькому роботу на базе Raspberry Pi 4  с 2 ГБ оперативной памяти и камерой на борту.
Цель этого проекта научить его распознавать дорожные знаки и двигаться при помощью камеры.

Материалы и комплектующие:
  1. Raspberry Pi 4 с 2 ГБ ОЗУ
  2. камера Raspberry Pi (но можно любую другую)
  3. Arduino Nano (Uno)
  4. моторы (мощность и тип по желанию)
  5. драйвер моторов (зависит от типа мотора)
  6. аккумуляторы на 12V (хочется на 2-3 Ач)
  7. гироскоп-акселерометр-компас (для определения угла поворота)
